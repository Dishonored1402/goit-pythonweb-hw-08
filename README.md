Цей застосунок — це REST API для керування контактами, побудований на FastAPI та SQLAlchemy. Основна увага приділена надійності даних та зручності використання.

На відміну від методу PUT, який вимагає надсилати всі поля, PATCH дозволяє змінювати лише конкретні параметри
У коді репозиторію використовується функція model_dump(exclude_unset=True). Вона ігнорує всі поля, які ви не вказали в запиті
При використанні PATCH потрібно залишати в JSON лише ті поля, які ви хочете змінити. Все інше потрібно видалити, щоб випадково не перезаписати існуючі дані значеннями за замовчуванням

Система автоматично перевіряє унікальність електронної пошти:

При спробі створити контакт із поштою, яка вже є в базі, сервер не видає технічну помилку 500
Замість цього він повертає зрозумілу відповідь 409 Conflict із повідомленням "Email already exists"

Реалізовано спеціальний маршрут /contacts/birthdays, який автоматично вираховує, у кого з контактів день народження відбудеться протягом наступних 7 днів
